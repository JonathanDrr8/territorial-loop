# Konzept — territorial-loop

> **Status:** Stub. Wird in der **Konzept-Phase (Phase A)** ausgefüllt.

## Kern-Idee

Ein Browser-basiertes Territorial-RTS mit einer fundamentalen Abweichung von
ähnlichen Spielen: **die Spielwelt hat keine Ränder**. Topologisch ist die Karte
ein Torus (oder evtl. Zylinder) — wandert man über den rechten Rand, erscheint
man wieder links. Das soll das Gefühl einer "echten Welt" ohne künstliche Grenzen
vermitteln und neue strategische Möglichkeiten eröffnen.
